# Dogri_Genre_Dataset
A multi-class Dogri document.
The dataset used in this study is a balanced Dogri text corpus consisting of 311 samples categorized into three genres: stories, poems, and news. Each instance includes a unique index, the textual content in Dogri, and its corresponding label. The dataset is evenly distributed across classes, ensuring unbiased model training and evaluation. It contains clean, non-null entries and is well-suited for supervised text classification tasks in low-resource language settings.
